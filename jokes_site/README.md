# Moroccan Jokes Website (نكت مغربية)

A premium-designed website that displays random Moroccan jokes.

## How to Run

Because this website loads data from a JSON file, modern browsers might block the request if you just double-click `index.html` (Cross-Origin Resource Sharing policy).

### Recommended Method (Local Server)
1. **VS Code Live Server**: Right-click `index.html` and select "Open with Live Server".
2. **Python**:
   Open a terminal in this folder and run:
   ```bash
   python -m http.server
   ```
   Then open `http://localhost:8000`.

## Features
- Premium Glassmorphism Design
- 100+ Moroccan Jokes
- Mobile Responsive
- Arabic Font Support (Cairo)
